Bash Cheat Sheet

Common Bash Commands

Command Option Description
ls -a list all, do not ignore entries starting with .
ls -l use a long listing format
history -c clear the history list by deleting all of the entries
history -d offset delete the history entry at position OFFSET.
kill terminate stalled processes
alias alias_name = “command_to_run”
grep -i search/ignore case distinctions in patterns and data
grep -n search/print line number with output lines
pwd print working directory
echo prints text to the terminal window
